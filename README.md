# squash-booking-automation

Python + Playwright automation for squash booking on Resamania.

## Features

- automatic login
- direct planning URL access
- court priority management
- lazy loading handling
- configurable waits
- dry-run mode

## Installation

```bash
pip install -r requirements.txt
python -m playwright install chromium
```

## Usage

```bash
python reserver_squash.py --dry-run --headed --debug
```
